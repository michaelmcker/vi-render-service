# Win/loss recap

Pull the transcript at $1 (or search Granola notes by title) and produce:

1. **Outcome**: won / lost / ongoing — with explicit signal from the transcript.
2. **Reasons** (3-5 bullets): use the prospect's own words wherever possible.
3. **What we did well**: ≤ 3 bullets.
4. **What we'd change next time**: ≤ 3 bullets.
5. **Suggested follow-up email** in Nikki's voice (use prompts/draft_reply.md
   style rules) — short, confirms next step or politely closes the loop.

Save to `~/hermes/output/winloss/{slug}-{date}.md`. Also create a Salesforce
note via Zapier with a 3-bullet summary (only if win/loss is conclusive).
